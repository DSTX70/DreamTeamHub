# What’s new

## 1) Bulk Import API

New endpoint in FastAPI:

* `POST /roles/import` — accepts a JSON array of RoleCard objects (same fields as `POST /roles`), creates them in one shot, returns created records.

Visible in your `/docs` once you run the backend.

**Example import (from seed JSON)**

```bash
# Start API first, then:
curl -X POST http://localhost:8000/roles/import \
  -H "content-type: application/json" \
  --data-binary @rolecards_seed.json
```

# How to use now

1. **Run backend**

```bash
cd backend
pip install -r requirements.txt
python -c "from app.seed import seed; seed()"
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

2. **Import the roster**

```bash
curl -X POST http://localhost:8000/roles/import \
  -H "content-type: application/json" \
  --data-binary @rolecards_seed.json
```

3. **Browse**

* Go to `/docs` → **roles** tag → list, filter by `pod`, or update entries.
* (Optional) Build a **Roster** UI page that lists cards from `GET /roles?pod=Marketing`.
