✅ **Repo/Replit:** **DreamTeamHub**
Below is a **paste-ready Replit drop** that adds **Default Cast** + **Curated Cast** controls to the **Strategy Session detail page**, and shows a **cast preview** on the **Strategy Sessions list**.

### What this adds

* **Default Cast** button (one click sets the standard invited cast)
* **Curated Cast** selector (multi-select from an expanded roster)
* **Participants saved on the Strategy Session** (`participants` field) via the existing `PUT /api/strategy-sessions/:id`
* List page shows **Cast count + preview** (first 3 names)

No backend changes required.

## Post-apply verification checklist

1. Open `/strategy/:id`
2. See **Cast (Participants)** card
3. Click **Use Default Cast** → selected chips update
4. Click **Curate Cast** → select/deselect names
5. Click **Save** → refresh page → cast persists
6. `/strategy` list shows cast count + preview badges
