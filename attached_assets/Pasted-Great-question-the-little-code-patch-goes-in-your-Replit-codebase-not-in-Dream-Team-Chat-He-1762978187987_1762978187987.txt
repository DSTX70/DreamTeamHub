## Where to paste the code

In Replit (your DTH app):

1. Open `app/client/pages/chat/ChatPage.tsx`
   (If your file is named differently, open the main chat page component.)
2. Make these three small edits:

### A) Update URL when a conversation is selected

Paste inside your `onSelectConversation(id)` (or wherever you set the active convo):

```ts
function onSelectConversation(id: string | number) {
  setSelectedId(String(id)); // your existing selection
  const url = new URL(window.location.href);
  url.searchParams.set('c', String(id));
  window.history.replaceState({}, '', url);
}
```

### B) Open a conversation from the URL on load

Near the top of the component:

```ts
import { useEffect } from 'react';

useEffect(() => {
  const c = new URLSearchParams(window.location.search).get('c');
  if (c) setSelectedId(String(c));
}, []);
```

### C) Add a “Copy link” button in the active conversation header

Where you render the current convo header:

```tsx
<button
  className="text-xs border rounded px-2 py-1"
  onClick={() => {
    const url = new URL(window.location.href);
    url.searchParams.set('c', String(selectedId));
    navigator.clipboard.writeText(url.toString());
  }}
>
  Copy link
</button>
```

3. Save → rebuild/restart your dev server if needed.
