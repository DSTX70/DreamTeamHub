## 1. Overview

This document defines the monetization strategy and packaging tiers for Dream Team Hub capabilities, including Agent Lab access, persona utilization, and advanced features.

---

## 2. Core Tiers

### 2.1 Starter (Free)
- **Agent Mode:** Full access (execution-only)
- **Dream Team Personas:** Read-only access to registry
- **Autonomy:** L0 only
- **Pack Generation:** 5 packs/month
- **Support:** Community

### 2.2 Pro
- **Agent Mode:** Full access
- **Dream Team Personas:** Up to 10 active personas
- **Agent Lab:** L1 autonomy (Suggest)
- **Pack Generation:** 50 packs/month
- **Export:** Drive integration enabled
- **Support:** Email

### 2.3 Business
- **Agent Mode:** Full access
- **Dream Team Personas:** Unlimited active
- **Agent Lab:** L1-L2 autonomy (Suggest, Act)
- **Pack Generation:** 500 packs/month
- **Export:** All integrations
- **Governance:** Full RACI and audit trail
- **Support:** Priority

### 2.4 Enterprise
- **Agent Mode:** Full access + custom capabilities
- **Dream Team Personas:** Custom persona creation
- **Agent Lab:** L1-L3 autonomy (full spectrum)
- **Pack Generation:** Unlimited
- **Export:** API access + white-label
- **Governance:** Custom policies + OS Council access
- **Support:** Dedicated + SLA

## 4. Pricing Model

### 4.1 Seat-Based
- Per-operator pricing for certified Agent Lab users
- Bulk discounts at 10+, 50+, 100+ seats

### 4.2 Usage-Based
- Pack generation overage at per-pack rate
- Agent Lab run-hours metered for L2+ operations

### 4.3 Hybrid
- Base subscription for tier features
- Usage add-ons for variable consumption

---

## 5. Upgrade Paths

| From | To | Trigger |
|------|-----|---------|
| Starter | Pro | Pack limit reached |
| Pro | Business | L2 autonomy needed |
| Business | Enterprise | Custom persona or API required |

---

## 6. Governance

- Tier changes require billing admin approval
- Usage metrics available in Ops Dashboard
- Downgrade policies protect data for 90 days
- Enterprise contracts negotiated individually
